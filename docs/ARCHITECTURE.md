# ARCHITECTURE

## High-Level System Overview
- **Frontend**: React Native, Expo, TypeScript, Zustand (for state management)
- **Backend**: Node.js, Express, Supabase (for database/auth)
- **Infrastructure**: EAS Update (for OTA), Render (for backend hosting), GitHub (for version control)
- **AI Integration**: NVIDIA Llama, Gemini, OpenRouter

*Note: Update this document as architectural boundaries and patterns evolve.*

# HumanOS North Star (Architecture Vision)
HumanOS is designed to be a Personal AI Operating System rather than a standalone app. Nova is simply the first application powered by this core.
